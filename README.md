## 使用技術

- 資料處理：Python pandas

## 作品說明
- 台灣地圖：可縮放平移、導覽至各縣市資料
- 投票資訊：可查詢全台各縣市、鄉鎮市區、村里
- 可視化圖表：歷屆政黨得票總覽，hover 顯示詳細資訊
